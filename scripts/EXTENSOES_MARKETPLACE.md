# 🔌 EXTENSÕES RECOMENDADAS PELO BRIAN (Marketplace)

Henrique, para acelerar nosso desenvolvimento e manter o padrão premium:

1. **Tailwind CSS IntelliSense**: Essencial para nossos estilos rápidos e responsivos.
2. **Prettier - Code formatter**: Garante que o código esteja sempre limpo e organizado.
3. **ESLint**: Para evitar bugs bobos antes mesmo de rodarmos o código.
4. **Material Icon Theme**: Melhora a visibilidade das pastas e arquivos.
5. **GitLens**: Para rastrearmos cada mudança estratégica.
6. **Error Lens**: Mostra os erros diretamente na linha de código.

> [!TIP]
> Instale estas extensões clicando no ícone de "Quadrados" (Extensions) no menu lateral esquerdo do VS Code.
