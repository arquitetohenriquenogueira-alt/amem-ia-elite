# Amém IA - Manual de Fluxo Profissional 🚀

Este guia ensina como usar suas ferramentas pagas para produzir 1 vídeo viral em menos de 5 minutos.

## 1. Fase de Inteligência (Gemini + Bridge)
- Use o script `content_bridge.py` para gerar o texto do devocional.
- Se não tiver ElevenLabs, o script salvará apenas o arquivo de texto em `G:\EMPRESA_DIGITAL\AME_IA\MARKETING\ROTEIROS`.

## 2. Fase de Design (Canva FREE)
- Abra o Canva (pode ser a conta gratuita).
- Use elementos gratuitos em Black/Gold.
- Exporte as imagens para o CapCut.

## 3. Fase de Finalização (CapCut PRO)
- Importe as imagens do Canva.
- Use a ferramenta **"Texto para Fala" (FREE)** do CapCut para narrar o roteiro gerado pelo Gemini.
- O CapCut Pro (que você já tem) fará as legendas dinâmicas e filtros premium.

## 4. Fase de Distribuição (TikTok/Reels)
- Poste seguindo os horários do nosso **Calendário de 30 Dias**.
- Use o CTA: "Link na Bio para falar com o Pastor IA".
