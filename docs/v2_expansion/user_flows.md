# 🗺️ FLUXOS DE USUÁRIO (USER FLOWS): Amém IA v2.0

Este documento mapeia as jornadas principais para garantir uma experiência integrada.

---

## 🛤️ Jornada 1: O Check-in Emocional
1.  **Entrada:** Usuário abre o app.
2.  **Ação:** Clica em "Como está se sentindo hoje?".
3.  **Processo:** Seleciona um sentimento (ex: Ansioso).
4.  **IA:** O sistema filtra o banco de `devotionals` por `emotion_tag`.
5.  **Resultado:** Abre o "Devocional do Dia" personalizado.
6.  **Next Step:** Opção de postar uma promessa bíblica no "Muro dos Testemunhos" (Rede do Fiel).

---

## 🛤️ Jornada 2: Da Célula para a Rede
1.  **Entrada:** Usuário acessa o menu "Mais" -> "Célula".
2.  **Ação:** líder registra a presença da reunião.
3.  **Processo:** O sistema detecta um "Crescimento" (novos visitantes).
4.  **Notificação:** IA sugere que o líder publique um agradecimento na "Rede do Fiel".
5.  **Resultado:** Visibilidade e engajamento da igreja local na plataforma global.

---

## 🛤️ Jornada 3: Pedido de Oração Ativo
1.  **Entrada:** Usuário clica em "Pedidos de Oração".
2.  **Ação:** Escreve um pedido.
3.  **IA:** O sistema sugere um versículo de conforto imediato (Mentor IA).
4.  **Processo:** O pedido aparece no feed da "Rede do Fiel".
5.  **Interação:** Outros usuários clicam em "Estou Orando".
6.  **Feedback:** O autor recebe notificação: "Aria e mais 5 pessoas estão orando por você".

---

## 📐 Mapa de Navegação
`Home` -> `Feeling Check` -> `Devocional` -> `Social Feed`
`Menu Mais` -> `Células (Liderança)` -> `Relatórios`
`Menu Mais` -> `Rede do Fiel` -> `Testemunhos/Pedidos`
